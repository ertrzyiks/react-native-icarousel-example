# react-native-icarousel-example

Example project with [react-native-icarousel](https://github.com/ertrzyiks/react-native-icarousel).

## Usage

```
<Carousel style={styles.container} type={Carousel.Type.iCarouselTypeWheel}>
  <Text> One! </Text>
  <Text> Two! </Text>
  <Text> Three! </Text>
  <Text> Four! </Text>
  <Text> Five! </Text>
</Carousel>
```
